How It Works

API Architecture Phase 1

01. Institution Calls API

POST /verify-pqc-signature with ML-DSA signature and metadata. RESTful endpoint. Standard API contract.

02. Backend Validates

Qubex backend validates ML-DSA-87 signature. Returns verification certificate. Response in milliseconds.

03. Institution Uses Certificate

Institution posts certificate on-chain as proof of PQC verification. Zero on-chain latency impact. Institution adopts standard.
